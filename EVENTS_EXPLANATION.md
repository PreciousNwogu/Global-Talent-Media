# Are the Events Real or Placeholders?

## Answer: **PLACEHOLDERS / SAMPLE DATA**

The events created by the seeder are **not real events**. They are **placeholder/sample data** used for development and testing purposes.

### What Makes Them Placeholders:

1. **Sample Content**:
   - Generic titles like "London Music Festival 2026"
   - Fictional descriptions
   - Made-up venue addresses

2. **Stock Images**:
   - Uses Unsplash (free stock photo service)
   - Generic music/tech/food images
   - Not actual event photos

3. **Sample Videos**:
   - YouTube embed URLs pointing to generic videos
   - Not actual event promotional videos

4. **Fictional Data**:
   - Prices are examples (£75, £299, etc.)
   - Dates are calculated (2 months from now, etc.)
   - Capacities are arbitrary numbers

5. **London Locations**:
   - Uses real London venue names (Hyde Park, Tate Modern, etc.)
   - But these are NOT actual bookings or events at those locations
   - Just using recognizable names for realism

### To Use Real Events:

You need to:
1. **Replace the seeder data** with your actual event information
2. **Upload real event images** (or use actual image URLs)
3. **Add real promotional videos** (or remove video_url if not needed)
4. **Update with actual dates, prices, and details**
5. **Use real venue information**

The current events are just examples to demonstrate how the system works!
